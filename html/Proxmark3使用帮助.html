    连接电脑

    官方

    Windows

    Linux

    克隆 proxmark3 库

    git clone https://github.com/proxmark/proxmark3.git

    发行版 安装依赖
    Kali Linux sudo apt install git build-essential libreadline5 libreadline-dev gcc-arm-none-eabi libusb-0.1-4 libusb-dev libqt4-dev ncurses-dev perl pkg-config libpcsclite-dev pcscd
    Ubuntu sudo apt install p7zip git build-essential libreadline5 libreadline-dev libusb-0.1-4 libusb-dev libqt4-dev perl pkg-config wget libncurses5-dev gcc-arm-none-eabi libstdc++-arm-none-eabi-newlib libpcsclite-dev pcscd
    Gentoo Linux sudo emerge git p7zip make gcc readline dev-libs/libusb perl wget dev-qt/qt-meta:4 sys-apps/pcsc-lite
    openSUSE sudo zypper install -t pattern devel_basis
    sudo zypper install git libreadline7 libusb-1_0-0 libusb-1_0-devel perl libqt4-devel readline7-devel termcap wget pcsc-lite

    冰人

    Windows

    详见Windows安装说明

    Linux

    克隆 RRG/proxmark3 库

    git clone https://github.com/RfidResearchGroup/proxmark3.git

    发行版 安装依赖 蓝牙支持（rdv4） 图形组件（hw tune）
    Debian sudo apt-get install --no-install-recommends git ca-certificates build-essential pkg-config libreadline-dev gcc-arm-none-eabi libnewlib-dev qtbase5-dev libbz2-dev libbluetooth-dev libbluetooth-dev qtbase5-dev
    ArchLinux sudo pacman -Sy git base-devel readline bzip2 arm-none-eabi-gcc arm-none-eabi-newlib qt5-base bluez --needed bluez qt5-base
    Ferdora sudo dnf install git make gcc gcc-c++ arm-none-eabi-gcc-cs arm-none-eabi-newlib readline-devel bzip2-devel qt5-qtbase-devel bluez-libs-devel libatomic bluez-libs-devel qt5-qtbase-devel
    OpenSUSE sudo zypper install git patterns-devel-base-devel_basis gcc-c++ readline-devel libbz2-devel cross-arm-none-gcc9 cross-arm-none-newlib-devel libqt5-qtbase-devel 不提供 libqt5-qtbase-devel

    根据自己的PM3平台进行判断

    如果使用的是rdv4
    可以直接编译make clean && make all
    也可以自己指定平台为PM3RDV4进行编译make clean && make all PLATFORM=PM3RDV4

    其他平台（例如我用的proxmark3 easy）  
    则指定平台为PM3OTHER进行编译
    make clean && make all PLATFORM=PM3OTHER

    需要具体调整固件的可以查看高级编译参数

    自行编译

    安装依赖

    pkg install make clang clang++ readline libc++ git

    克隆 RRG/proxmark3 库

    git clone https://github.com/RfidResearchGroup/proxmark3.git

    只编译客户端

    cd proxmark3 && make clean && make client
    有root权限  

    在Termux安装tsu 

    pkg install tsu

    sudo client/proxmark3 /dev/ttyACM0 //

    无root权限

    安装TCPUART

    从上往下

    USB_USRT
    点击Connect
    弹窗“要允许TCPUART访问proxmark3吗”点确定

    TCP
    切换 client -> server
    修改端口（默认8080）
    点击Start  

    然后Termux处输入

    client/proxmark3 tcp:localhost:端口

    官方固件命令使用帮助

    hw 硬件检测相关命令

    命令 解释
    detectreader [‘l’|’h’] – Detect external reader field (使用选项 ‘l’ 或 ‘h’ 来限制 低频 或 高频)
    fpgaoff Set FPGA off
    lcd <HEX command> <count> – Send command/data to LCD
    lcdreset Hardware reset LCD
    readmem [address] – Read memory at decimal address from flash
    reset Reset the Proxmark3
    setlfdivisor <19 - 255> – Drive LF antenna at 12Mhz/(divisor+1)
    setmux <loraw|hiraw|lopkd|hipkd> – Set the ADC mux to a specific value
    tune [‘l’|’h’] – 测量天线调谐 (使用选项 ‘l’ 或 ‘h’ 来限制 低频 或 高频)
    version Show version information about the connected Proxmark
    status Show runtime status information about the connected Proxmark
    ping 测试pm3是否有响应

    hw tune //测量天线调谐

    hw version //显示Proxmark3的固件版本
